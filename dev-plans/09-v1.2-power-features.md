# Plan 09: v1.2 — Power Features

**Estimated effort:** 6–8 days
**Depends on:** Plan 08 (v1.1)
**Phase:** v1.2 (PRD Phase 3)

## Objective

Add power-user features: sleep timer, dyslexia-friendly font overlay, advanced ElevenLabs voice controls, settings export/import, custom CSS selectors per domain, and a Firefox port. After this plan, the extension covers all P1 and P2 features from the PRD.

## Tasks

### 1. Sleep timer (`/src/lib/sleep-timer.ts`)

- User sets a timer (5, 10, 15, 30, 60 min, or custom) — FR-11
- Timer runs in the service worker via `chrome.alarms`
- On expiry: fade audio volume to 0 over 5 seconds, then pause
- UI: timer selector in popup + toolbar expanded state
- Configurable default timer duration in settings

### 2. Dyslexia-friendly font overlay (`/src/content/font-overlay.ts`)

- Inject a toggleable overlay that changes page text rendering — FR-45:
  - Font: OpenDyslexic or Lexie Readable (bundled as web fonts in the extension)
  - Size boost: user-configurable (+2px, +4px, +6px)
  - Line height boost: 1.5×, 1.75×, 2×
  - Letter spacing boost
- Apply via injected `<style>` targeting the extracted article container
- Toggle from toolbar or settings
- Settings: font choice, size boost, line height — persisted in `chrome.storage.local`

### 3. Advanced ElevenLabs voice parameters (`/src/options/VoiceAdvanced.tsx`)

- Expose `voice_settings` in an "Advanced" panel when ElevenLabs is active — FR-33:
  - Stability slider (0–1)
  - Similarity Boost slider (0–1)
  - Style slider (0–1)
- Save per-voice settings in storage
- Pass through to the ElevenLabs adapter's `synthesize()` call

### 4. Voice preview (`/src/options/VoicePreview.tsx`)

- "Preview" button next to each voice in the settings voice list — FR-34
- Synthesizes a 5-second sample text ("The quick brown fox...") using the selected voice
- Plays in the options page via a simple `<audio>` element
- Loading state + error handling

### 5. Export / Import settings (`/src/options/ExportImport.tsx`)

- **Export** — FR-20:
  - Read all `chrome.storage.local` data
  - Redact API keys (replace with `"REDACTED"`)
  - Download as `immersive-reader-settings.json`
- **Import**:
  - Upload JSON file
  - Validate schema
  - Merge with existing settings (or full replace with confirmation)
  - Keys remain redacted — user must re-enter keys after import
- UI: buttons in Settings → Advanced

### 6. Custom CSS selectors per domain (`/src/lib/site-selectors.ts`)

- Advanced users define a CSS selector to pin content extraction — FR-25
  - e.g., for `example.com`, use `article.post-content`
- Storage: `{ [domain]: cssSelector }` in `chrome.storage.local`
- Content extractor checks for a domain-specific selector before falling back to Readability
- UI: manage rules in Settings → Advanced (add/edit/remove)

### 7. Skip references in academic papers (`/src/content/extractor.ts`)

- Detect reference/bibliography sections — FR-28:
  - Heuristic: heading containing "References", "Bibliography", "Works Cited"
  - Or: ordered list at end of document with citation patterns
- Toggle: on/off in settings (default: on)
- When enabled, chunker stops before the references section

### 8. Firefox port (stretch goal)

- Audit `manifest.json` for Firefox MV3 compatibility
- Replace Chrome-specific APIs:
  - `chrome.offscreen` → Firefox equivalent or polyfill
  - `chrome.storage` API is compatible
  - `chrome.commands` → `browser.commands`
- Use `webextension-polyfill` for cross-browser compatibility
- Test on Firefox Nightly with MV3 support
- Submit to Firefox Add-ons (AMO)

## Exit Criteria

- [ ] Sleep timer pauses playback after set duration with volume fade
- [ ] Dyslexia-friendly font overlay applies OpenDyslexic with configurable sizing
- [ ] ElevenLabs stability/similarity/style sliders affect synthesis output
- [ ] Voice preview plays a 5-second sample for any voice
- [ ] Settings export downloads a valid JSON file with keys redacted
- [ ] Settings import restores preferences (minus keys) correctly
- [ ] Custom CSS selectors override Readability extraction on configured domains
- [ ] Reference sections are detected and skipped in academic papers
- [ ] (Stretch) Extension loads and functions on Firefox
